后端管理插件hexo-admin 

1
2
3

npm install --save hexo-admin
hexo server -d
open http://localhost:4000/admin/

设置后台密码 修改站点配置文件，就是网站根目录下的 _config.yml文件:

1
2
3
4

admin:
  username: finger
  password_hash: $2a$1$Cof3VuvY8nKKIUjeCNBSE.HjcrCKQ1P80GEegP//SLDFWZoGzm4pa
  secret: a secret something
  • username：后端登录用户名
  • password_hash：后端登录用户密码对应的md5 hash值
  • secret：用于保证cookie安全

密码生成 hexo-admin密码是bcrypt编码。因此需要安装bcrypt-nodejs模块

1
2
3
4

$ node
> const bcrypt = require('bcrypt-nodejs')
> bcrypt.hashSync('your password secret here!')
//=> '$2a$10$8f0CO288aEgpb0BQk0mAEOIDwPS.s6nl703xL6PLTVzM.758x8xsi'

在线生成 https://www.bcrypt-generator.com/
